# 📊 Superstore Power BI Dashboard

## 🚀 Project Overview

This project presents an interactive Business Intelligence dashboard built using Power BI to analyze Superstore sales data. The dashboard enables stakeholders to monitor key performance indicators (KPIs), evaluate regional performance, analyze customer segments, and make data-driven business decisions.

---

## 🎯 Project Objectives

* Monitor overall sales performance
* Analyze regional sales distribution
* Evaluate category-wise sales performance
* Understand customer segment contributions
* Provide interactive filtering and reporting capabilities
* Support business decision-making through visual analytics

---

## 🛠️ Tools & Technologies

* Microsoft Power BI

* Data Modeling
* Business Intelligence Reporting

## 💡 Business Insights

* West region achieved the highest sales performance.
* Consumer segment contributed the largest share of revenue.
* Technology category generated the highest sales.
* Sales performance varied significantly across states.
* Interactive filtering enables detailed business analysis.
